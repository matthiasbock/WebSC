WebSC
=====

Python-based standalone web service, created using the Python-Bobo framework.

Functions:
* create yourself a private/public key pair
* or use the key pair from your smartcard
* create yourself a browser certificate (for automatic login)
* revoke your key pair
* manage the public keys of your associates
* sign documents and keys or revoke signatures
* check the validity of document signatures
* encrypt documents for your associates
* decrypt document, which your associates send to you
